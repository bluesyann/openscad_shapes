OpenScad Shapes
======
This repo contains various shapes designed with OpenScad. I try to keep them as parametric as possible, but if you notice bugs or unusual behaviour I would be happy to get your feedback or your PR!

Build plate holder
-----------
Current parameters are for Anycubic Photon Mono 4K 3d printer. I guess it can be adapter to similarly-designed printers.
There is no thread on the four sides-holes because I use two threaded rods going all the way through. Inserts could also be used with wider holes.

Center finder
-----------
Another device allowing to easily draw the central axis of an object up to 7 cm wide.
This design is based on a dual rack-and-pinion movig in opposite directions, the pinion staying at the center of the object.

Pipe fittings
-----------
Parametric library for generating any kind of planar pipe fittings. I'm using them for my plant-watering system, I guess it can also be used for aquariums, water-cooling etc.
